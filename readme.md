### Backend DEv

1. Create git repo -
   git init

2. Add a .gitignore file

3. Create .env file for environment variables and read data with dotenv -
   npm i dotenv

4. Generate package.json -
   npm init -y

5. create express app - 
   npm i express

6. create server.js

7. Connect to database - 
   npm i mongoose

8. Inialize PORT,MONGO_URL in .env and import it in server.js

9. ADd middlewares - body parser, err mid

10. Design schemas and create models

11. Design REST APIs for all resources
